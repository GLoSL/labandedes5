Compréhension de la fonction de chaque class id etc.....

les div portants une ID ne sont autres que les divs personelles, propres à chaque histoire et donc personnes.

les class

lesconteneursdimages: class affectant tout les conteneur de toute les histoire

"nomdelapersonne"+ imageconteneur: class affectant tout les conteneurs d'histoire de la personne
encore plus précis: "nomdelapersonne"+ imageconteneur + numéro du conteneur: permet de cibler un conteneur en particulier

paysages: class affectant toute les image paysage de toute les histoires

variante de la class paysages:
paysages + "nomdelapersonne": class affectant toute les images paysage de la personne en question
pour plus de précision il y'a encore paysage + "nom de la personne" + 1 2 ou 3 ou plus selon son numéro, ceci permet de cibler plus précisément une class paysage

portraits: class affectant toute les image portrait de toute les histoires
cf la l'explication sur la class paysage mais en remplaçant paysage par portrait

images: class affectant toute les images de tout le site

imagehistoire: class affectant toute les images de toute les histoire

"nomdelapersonne" + image: exemple okbaimage

"nomdelapersonne" + image + un numéro: exemple okbaimage1
